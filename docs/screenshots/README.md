# Dashboard Screenshots

Take these screenshots from the running dashboard (`make serve`) and save them in this
folder with these exact filenames:

| Filename | What to capture |
|---|---|
| `01_forecast_tab.png` | Full Forecast tab with **All Stores** selected |
| `02_store_forecast.png` | Forecast tab with a **single store** selected (e.g. Store 1) |
| `03_model_comparison.png` | Model Comparison tab showing the metrics table |
| `04_ensemble_weights.png` | Ensemble weights chart on the Model Comparison tab |
| `05_data_explorer.png` | Data Explorer tab |
| `06_shap_global.png` | SHAP global feature importance chart |
| `07_decomposition.png` | Seasonal decomposition chart |

After saving the screenshots, add them to `README.md` in the Results section using:

```markdown
![Forecast Tab](docs/screenshots/01_forecast_tab.png)
![Model Comparison](docs/screenshots/03_model_comparison.png)
![SHAP Importance](docs/screenshots/06_shap_global.png)
```
